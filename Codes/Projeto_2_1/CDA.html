Churn Data Analysis

The turnover rate, in its broadest sense, is a measure of the number of individuals or items that leave a collective group during a specific period. It is one of the two main factors that determine the steady level of customers a company will support.

Through this, the analysis of these churn data ends up becoming very relevant in today's companies, and therefore, I tried to simulate as much as possible a similar situation of a telecommunications company.

Development

For the development of the project, pandas libraries were used to manipulate the data, and plotly to create the histogram.

Conclusion

Finally, it is possible to obtain a spreadsheet with a good ETL treatment, and it is also possible to use histograms for better decision-making by the team.

Source Code

import pandas as pd

#Understand how the Churn values are distribuited between the clients

# Step 1: Import database

dataset=pd.read_csv("telecom_users.csv")

# Step 2: Visualize database
#display(dataset)

# Passo 3: Data processing(Correct database problems)
# Useless column
# axis = 0 -> line
# axis = 1 -> column
dataset=dataset.drop("Unnamed: 0", axis=1) #DROP request (WHAT, AXIS=COLUMN_OR_LINE)
display(dataset)


#If "lira" induces to 'error', so we gonna turn it to NULL
dataset["TotalGasto"]=pd.to_numeric(dataset["TotalGasto"], errors='coerce')
#'coerce' is the word that indicates NULL

print(dataset.info())

# Delete NULL values
#all -> deletes all the NULL columns
#any -> deletes the columns that got at least one NULL value

dataset=dataset.dropna(how="all", axis=1) #Deletes NULL columns

dataset=dataset.dropna(how="any", axis=0) #Deletes the lines whose got at least one NULL value

print(dataset.info())

# Step 4: Inicial Analysis

print(dataset["Churn"].value_counts())
print(dataset["Churn"].value_counts(normalize=True).map("{:.1%}".format))

# Normalize gets us the percentual value

# Step 5: Detailed client Analysis
# https://plotly.com/python/histograms/ to set the colors
import plotly.express as px

grafico = px.histogram(dataset, x="MesesComoCliente", color="Churn")
grafico.show()
