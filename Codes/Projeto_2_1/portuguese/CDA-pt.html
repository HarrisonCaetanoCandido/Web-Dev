Análise de Dados de Churn

A taxa de rotatividade, em seu sentido mais amplo, é uma medida do número de indivíduos ou itens que deixam um grupo coletivo durante um período específico. É um dos dois principais fatores que determinar o nível estável de clientes que uma empresa atenderá.

Com isso, a análise desses dados de churn acaba se tornando muito relevante na atualidade empresas e, portanto, tentei simular ao máximo uma situação semelhante de um empresa de telecomunicações.

Desenvolvimento

Para o desenvolvimento do projeto, as bibliotecas usadas foram pandas, responsável por manipular os dados, e a plotlt, que cria histrogramas e outros gráficos.

Conclusão

Por fim, foi possível obter uma base de dados com um bom tratamento na ETL e também a geração de gráficos para uma melhor tomada de decisão pelo time de vendas.

Código Fonte

import pandas as pd

#Understand how the Churn values are distribuited between the clients

# Step 1: Import database

dataset=pd.read_csv("telecom_users.csv")

# Step 2: Visualize database
#display(dataset)

# Passo 3: Data processing(Correct database problems)
# Useless column
# axis = 0 -> line
# axis = 1 -> column
dataset=dataset.drop("Unnamed: 0", axis=1) #DROP request (WHAT, AXIS=COLUMN_OR_LINE)
display(dataset)


#If "lira" induces to 'error', so we gonna turn it to NULL
dataset["TotalGasto"]=pd.to_numeric(dataset["TotalGasto"], errors='coerce')
#'coerce' is the word that indicates NULL

print(dataset.info())

# Delete NULL values
#all -> deletes all the NULL columns
#any -> deletes the columns that got at least one NULL value

dataset=dataset.dropna(how="all", axis=1) #Deletes NULL columns

dataset=dataset.dropna(how="any", axis=0) #Deletes the lines whose got at least one NULL value

print(dataset.info())

# Step 4: Inicial Analysis

print(dataset["Churn"].value_counts())
print(dataset["Churn"].value_counts(normalize=True).map("{:.1%}".format))

# Normalize gets us the percentual value

# Step 5: Detailed client Analysis
# https://plotly.com/python/histograms/ to set the colors
import plotly.express as px

grafico = px.histogram(dataset, x="MesesComoCliente", color="Churn")
grafico.show()
